Comic Cav // API practice


This was my first ever project i was able to take on and with my sights set i went for a project where i could focus on one pivital point which is used in every website... API. Completing this project allowed me to learn the process of how to pull information of an API through the inspect tool and pull information and lines of code to create an api sectino in my "practice website."

I was able to achieve this through inspecting the running the api into the console log, pull information through said console and pick apart each aspect of the API card i wanted to display, the parts i pulled were the: image, comic issue, comic name and their release date. Thus when adding all of these aspects into its own styled cards, i was able to pull information of the latest comic releases and display them in blocks into front end on the website!
